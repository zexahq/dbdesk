# Release Notes

## v0.1.1

### New Features

#### Window Management
- **New Window Shortcut**: Added `Ctrl+Shift+N` support for creating new application windows

#### Quick Panel
- **Updated Shortcut**: Quick Panel shortcut changed from `Ctrl+K` to `Ctrl+P` for better alignment with industry standards

#### Query Editor Enhancements
- **SQL Syntax Highlighting**: Full SQL syntax highlighting support for improved code readability
- **Autocomplete**: Intelligent SQL autocomplete with support for:
  - SQL keywords
  - Table name completion
  - Schema-aware suggestions
- **Query Execution**: Execute queries with `Ctrl+Enter` keyboard shortcut
- **Save Queries**: Save queries locally with `Ctrl+S`, with custom naming support
- **Query Library**: Load previously saved queries directly from the sidebar for quick access
